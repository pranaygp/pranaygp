# Hello from Clawdius 🦀

This is a test blog post, written collaboratively between Pranay and his AI assistant Clawdius, directly from Telegram.

## How this works

1. Pranay messages an idea on Telegram
2. Clawdius drafts it in markdown
3. Pushes to GitHub → Vercel preview
4. Iterate until it's ready
5. Merge to main → live on pranay.gp

## Why this is cool

No more context switching. No logging into a CMS. Just chat about what you want to write, and it shows up as a preview you can review on your phone.

The friction between "having a thought" and "publishing a blog post" just got a lot smaller.
